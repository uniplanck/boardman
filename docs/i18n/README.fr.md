# Board-Man

[English](../../README.md) / [ja](README.ja.md) / [zh-CN](README.zh-CN.md) / [es](README.es.md) / [pt-BR](README.pt-BR.md) / [ko](README.ko.md) / [de](README.de.md) / [fr](README.fr.md)


Board-Man est un outil open-source de productivité pour le presse-papiers sur macOS, dérivé de Clipy.

Il ne sert pas seulement à conserver l’historique du presse-papiers. Son objectif est de rendre plus visibles les flux de travail basés sur la copie, le collage, l’édition et le déplacement de texte entre applications.

![Board-Man clipboard history](../images/boardman-history.png)

## Ce que fait Board-Man

Board-Man vise les flux de travail où les actions de copier-coller se répètent souvent au cours de la journée.

Axes principaux:

- consulter l’historique du presse-papiers
- rendre l’activité de collage plus visible
- améliorer les flux de travail centrés sur le copier-coller
- afficher l’état dans la barre de menus
- fonctionner comme un utilitaire local pour macOS

## Pour qui

- personnes qui écrivent
- développeurs
- opérateurs vidéo, réseaux sociaux et marketing
- utilisateurs qui manipulent souvent des textes modèles et des URL
- utilisateurs Mac ayant beaucoup de tâches de copier-coller
- personnes intéressées par l’automatisation locale et la visualisation du travail
