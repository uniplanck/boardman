# Board-Man

[English](../../README.md) / [ja](README.ja.md) / [zh-CN](README.zh-CN.md) / [es](README.es.md) / [pt-BR](README.pt-BR.md) / [ko](README.ko.md) / [de](README.de.md) / [fr](README.fr.md)


Board-Man es una utilidad de productividad para el portapapeles en macOS, de código abierto y derivada de Clipy.

No se centra solo en guardar el historial del portapapeles. Su objetivo es hacer más visible el trabajo basado en copiar, pegar, editar y mover texto entre aplicaciones.

![Board-Man clipboard history](../images/boardman-history.png)

## Qué hace

Board-Man está pensado para flujos de trabajo donde copiar y pegar ocurre muchas veces durante el día.

Áreas principales:

- revisar el historial del portapapeles
- hacer visible la actividad de pegado
- mejorar flujos de trabajo centrados en copiar y pegar
- mostrar estado desde la barra de menú
- funcionar como utilidad local para macOS

## Para quién es

- personas que escriben
- desarrolladores
- operadores de vídeo, redes sociales y marketing
- usuarios que manejan textos repetidos y URLs con frecuencia
- usuarios de Mac con mucho trabajo de copiar y pegar
- personas interesadas en automatización local y visibilidad del trabajo
