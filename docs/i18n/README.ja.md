# Board-Man

[English](../../README.md) / [ja](README.ja.md) / [zh-CN](README.zh-CN.md) / [es](README.es.md) / [pt-BR](README.pt-BR.md) / [ko](README.ko.md) / [de](README.de.md) / [fr](README.fr.md)

Board-Man は、Clipy から派生した macOS 向けのクリップボード生産性ツールです。

メニューバーからクリップボード履歴を呼び出せるだけでなく、コピー、貼り付け、編集、テキストや画像の移動が多い作業を見える化しやすくします。

![Board-Man main screenshot](../assets/board-man-main-screenshot.png)

## できること

- メニューバーから最近のクリップボード履歴を確認
- よく使う文章をスニペットとして保存して貼り付け
- 貼り付け回数バッジで、よく使う項目を確認
- 画像だけのクリップボード項目を含む画像クリップボードに対応
- クリップボード履歴の検索とキーボード操作
- ショートカット、メニュー表示、履歴件数、テーマ/ライト寄り表示の設定
- クリップボード内容を外部サービスへ送らず、ローカルで動作

## ダウンロード

- [Board-Man v1.2.3 をダウンロード](https://github.com/uniplanck/boardman/releases/tag/v1.2.3)
- macOS app archive: `Board-Man-v1.2.3.zip`

## インストールと初回起動

1. リリースページから `Board-Man-v1.2.3.zip` をダウンロードします。
2. zip を展開します。
3. `Board-Man.app` を `/Applications` に移動します。
4. Board-Man を起動します。

初回起動時に macOS Gatekeeper でブロックされる場合があります。その場合は **システム設定 > プライバシーとセキュリティ** から許可するか、アプリを Control クリックして **開く** を選んでください。

## 基本的な使い方

- メニューバーアイコンからクリップボード履歴を開きます。
- 履歴項目を選ぶと、現在のアプリへ貼り付けできます。
- 履歴が増えたら検索で絞り込めます。
- パネル内はキーボードで移動できます。
- 設定からショートカット、履歴件数、メニュー表示、テーマを調整できます。

## クリップボード履歴

Board-Man は、最近コピーしたテキスト、URL、画像などを履歴として保持します。もう一度コピーし直さなくても、必要な項目を選んで貼り付けられます。

## スニペット

スニペットは、よく使う文章、テンプレート、URLなどを保存して再利用するための機能です。スニペットエディタでフォルダや項目を作成し、メニューやショートカットから貼り付けできます。

## 貼り付け回数バッジ

貼り付け回数バッジにより、どの項目を何度使っているかを確認しやすくなります。文章作成、編集、開発、運用など、繰り返しの多い作業の把握に役立ちます。

## 画像クリップボード対応

Board-Man は画像のクリップボード項目にも対応しています。スクリーンショットや画像だけの項目を履歴から確認し、必要に応じて再利用できます。

## 検索とキーボード操作

検索で履歴を絞り込み、キーボードで項目を移動して貼り付けできます。作業中のアプリから意識を大きく移さずに使えることを重視しています。
