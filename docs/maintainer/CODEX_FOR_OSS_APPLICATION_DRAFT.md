# Codex for OSS Application Draft

## Short project description

Board-Man is an open-source macOS clipboard productivity utility derived from Clipy. It focuses on paste activity visibility and workflow-oriented clipboard usage for writers, developers, marketers, and operators who repeatedly move text across apps.

## Why this is useful

Clipboard activity is a core part of many real-world workflows, but most clipboard tools only store history. Board-Man explores clipboard usage as a productivity signal by making paste-heavy workflows easier to observe, improve, and eventually automate.
